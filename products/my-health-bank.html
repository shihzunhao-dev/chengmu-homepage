04SETUP — 20-40 分鐘從零裝好

20-40 分鐘從零裝好

裝 Python + 雙擊 install

有 Python 3.12 → 雙擊 install.bat → 自動裝 Flask、mkcert、產生 config.toml

產 HTTPS 憑證

跑 generate_cert.bat → mkcert 把 root CA 裝進信任區、產 LAN IP 憑證（PWA 必備）

取健保 token

用自然人憑證登入「我的健康存摺」→ F12 開 DevTools 複製 token → 貼到 Admin UI

建家庭成員 + 抓資料

Admin UI 新增成員（emoji 頭像 + 顏色）→ 按「抓 + 建置 + 翻譯」→ 等 LLM 跑完

手機加到主畫面

同網段手機開瀏覽器輸入 https://<你電腦的LAN IP>:8766 → 加到主畫面就能離線看

06FAQ — 常見問題

常見問題

需要什麼系統環境？

Windows 10 / 11 + Python 3.12 以上。家裡 Wi-Fi（手機 PWA 要跟電腦同網段才能連）。Mac/Linux 理論上也跑得起來但目前未官方支援。

健保 token 怎麼取得？要花錢嗎？

免費。用「自然人憑證」或「行動電話認證」登入 https://myhealthbank.nhi.gov.tw → 按 F12 → Application → Session Storage 複製 token 字串貼進來即可。token 約 30-60 分鐘過期，每次抓資料前要重新登入取新的。

LLM API 費用是分開的嗎？

是的。我們提供軟體本體與整合，LLM API 費由您自備帳號付。推薦 Groq（gpt-oss-120b 模型免費額度高，一般家庭一年用不完）。如不想用雲端 LLM，也可改 Ollama 跑本機模型（完全離線）。

每位家人都要分開取 token 嗎？

對。健保 token 是「個人」的，每位成員要用自己的自然人憑證登入取自己的 token。一旦各自取好，後面就只是「按按鈕」的事。

同一份報告會重複收 LLM 費嗎？

不會。系統用 SHA-256 比對病歷文字，翻過的存在 disk cache。即使換 LLM 服務商試效果，已翻過的都從快取讀，不再呼叫 API。

資料安全嗎？會上雲嗎？

所有健保原始資料只存在你電腦的 data/<家人ID>/ 資料夾。LLM 翻譯時會把英文片段傳到你選的服務商（OpenAI/Groq/Anthropic/Gemini），翻完只回字串、不留紀錄。如不想任何資料外流，可改用 Ollama 跑本機模型。

有試用版嗎？

14 天滿意保證：付款後 14 天內若不滿意全額退費（無條件）。沒有「免費試用」是因為產品價值在「持續累積家人病歷」，要真的抓幾位家人的資料才看得出價值。

付款方式？

銀行轉帳 / 個人匯款。確認入帳後 1 個工作天內寄出序號 + 軟體 zip + 安裝指引。如需開立公司發票（贈送長輩員工等用途）請於詢問時告知。
