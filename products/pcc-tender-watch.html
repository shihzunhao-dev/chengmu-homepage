06FAQ — 常見問題

常見問題

需要什麼系統環境？

Windows 10 Pro / 11 Pro / Server。一台不常關機的辦公室電腦當伺服器（不需要雲端 / VPS / Docker）。會自動啟用 Windows 內建的 IIS 功能。

同事在家裡 / 出差能看嗎？

標準版只能公司內網用。如需外網存取，可選客製版搭配 VPN / Cloudflare Tunnel / 或我們協助設定 HTTPS + 公網。

如果政府採購網改版，怎麼辦？

標準版不含改版維護（屬於工程性質的重新爬取邏輯設計）。進階版含 1 年維護，期間內政府網改版我們會優先處理。客製版可簽長期維護合約。

序號是什麼？綁定機器是怎麼回事？

每張序號永久授權，但一次只能在一台機器上啟用。換電腦（重灌 Windows / 換主機板 / 整顆硬碟搬機）需聯絡我們解綁原序號，新機器重新啟用即可。日常硬體變動（加裝記憶體、換 USB、更新 Windows）不影響。

有試用版嗎？

可申請 14 天滿意保證：付款後 14 天內若不滿意全額退費（無條件）。沒有「免費試用」是因為產品價值在「持續累積資料」，14 天才看得出效果。

跟坊間 SaaS 標案訂閱服務比，差別在哪？

(1) 一次買斷，沒有月費 (2) 資料完全在貴公司本機，不外流 (3) 不限同仁人數使用 (4) 沒有 SaaS 倒閉風險。但缺點：政府網改版時要付費維護（SaaS 通常自動處理）。適合「資料保密」與「長期使用」的中小企業；如果只是短期需求或不在意資料外流，SaaS 月租也是合理選擇。

為什麼要永久授權還要連雲端？

只有「啟用」階段需要連我們的雲端驗證（防止一張序號被多台機器盜用）。日常每天的更新與瀏覽完全離線。即使我們公司倒了，已啟用的軟體仍可繼續使用（只有「換新機器啟用」才會無法操作）。

付款方式？

銀行轉帳 / 公司對公司開發票（5% 稅外加）。確認入帳後 1 個工作天內寄出序號 + 軟體 zip。
